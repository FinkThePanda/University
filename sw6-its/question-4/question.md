# 4. Autentifikation på hjemmesider

- Redegør for og vis eksempler på hvordan man kan implementere autentifikation på hjemmesider samt web services, herunder basic authentication, cookies samt bearer tokens overordnet.
- Foklar hvilke sårbarheder man skal være særligt opmækrsomme på i forbindelse med autentifikation, herunder sikker kommunikation samt opbevaring af adgangskoder og øvrige sårbarheder som omtalt i OWASÅ A7 Authentication Failures.
