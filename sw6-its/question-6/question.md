# 6. OWASP Web Top 10: A1 og A2

- Redegør for hvad OWASP er og redegør for og vis eksempler på sårbarheder i følgende kategorier i OWASP Web Top 10 2025, samt forklar hvordan man sikre sig mod disse:
  - A1 Broken Acces Control (inkl. Cross-Site Request Forgery).
  - A2 Security Misconfigutaion
- Samt forklar fordele og ulemper ved automatiske sårbarhedsscannere.
